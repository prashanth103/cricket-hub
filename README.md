## Getting Started

### Prerequisites

* Node.js
* npm (or pnpm/yarn)

### Installation

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm install

# Start the development server
npm run dev
```

### Build for Production

```sh
npm run build
```

### Preview Production Build

```sh
npm run preview
```

## Project Structure

## Deployment

Build the project:

```sh
npm run build
```

The production files will be generated in the `dist/` directory and can be deployed to any static hosting provider such as:

* Vercel
* Netlify
* Cloudflare Pages
* GitHub Pages
* AWS S3 + CloudFront
